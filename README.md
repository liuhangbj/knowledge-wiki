## 核心框架

三维传导模型：

```
地缘政治冲突（美以伊战争 + 俄乌冲突）
    │
    ├─→ 能源供给冲击（霍尔木兹封锁 + 俄乌能源制裁）
    │       ├─→ 油价飙升（布伦特100-125美元）
    │       ├─→ 能源危机传导链（四层递进）
    │       ├─→ 天然气/LNG危机（卡塔尔20%产能停滞）
    │       └─→ 通胀与货币政策困境（二次通胀风险）
    │
    ├─→ 矿产供应链重构（制裁与反制裁）
    │       ├─→ 关键矿产依赖暴露（氦气、乙二醇、LPG、铝、硫磺）
    │       ├─→ 资源民族主义兴起（海峡收费制度化）
    │       └─→ 制造业成本曲线永久性抬升
    │
    └─→ 汇率与资本流动（战争溢价 + 货币韧性分层）
            ├─→ 美元流动性短缺（石油美元回流中断）
            ├─→ 美元指数走强与黄金反常下跌
            ├─→ 石油人民币临界点（41%结算占比）
            ├─→ 货币韧性分层（能源进口国受双重打击）
            └─→ 避险资产重定价（黄金、有色金属）
```

# 从烂尾的战争再认识中东的石油

## 核心论点

2026年美以伊战争虽呈"烂尾"状态，但已引发史上最严重的全球能源危机。霍尔木兹海峡封锁导致国际石油价格史上第四次站稳100美元/桶大关，阿联酋退出OPEC，IEA释放史上最大规模石油储备。文章通过详实数据论证：中东石油在全球能源格局中的不可替代性，以及能源转型热潮下世界仍离不开石油的现实。

## 关键信息

- **战争时间线**：2026年2月28日爆发 -> 4月7日暂停轰炸两周 -> 5月5日美国宣布"自由计划"暂停 -> 伊朗宣布成立"波斯湾海峡管理局"
- **油价数据**：布伦特从战前59.96美元涨至最高118.35美元（+63.28%）；WTI从67.02涨至112.95美元（+68.53%）；阿曼原油最高166.96美元/桶
- **海峡封锁影响**：约2000艘船只滞留波斯湾，2万名海员滞留；霍尔木兹原油流量从2000万桶/天骤降至200万桶/天
- **人员伤亡**：伊朗1348-2300死亡、1.7-3.3万受伤；美军13死、365伤；以色列5死、7656人送医
- **经济损失**：伊朗直接损失1440-1450亿美元，里亚尔贬值超30倍，GDP萎缩8.8%-10.4%
- **OPEC地震**：阿联酋2026年5月1日宣布退出OPEC及OPEC+，计划投资550亿美元提速石油项目
- **IEA行动**：3月11日宣布释放4亿桶石油储备，为史上第六次、规模最大的一次集体释放
- **IMF预测**：若战事持续至2027年、油价达125美元，全球经济增速恐降至约2%，通胀达5.8%

## 提取的概念
- [[霍尔木兹海峡]] — 国际油气贸易咽喉要道，战争中被伊朗封锁并建立收费制度
- [[石油美元体系]] — 1974年建立，当前因石油供应中断面临回流机制动摇
- [[能源危机]] — 史上规模最大供应中断，日减1300万桶，超过前三次危机总和
- [[战争溢价]] — 地缘政治风险对油价的持续性推升效应

# 高盛：参考1990年石油危机，美联储终将降息

## 核心论点

高盛认为市场对油价驱动的通胀反应过度，错误定价了美联储加息的可能性。参照1990年石油危机的历史先例，供给侧冲击下美联储最终更可能降息而非加息。当前通胀由油价推动，属于供给侧冲击，而非需求侧过热，美联储通常会无视此类价格压力。

## 关键证据

1. **市场定价的鹰派转向**：近期市场因油价上涨和滞胀担忧，从押注降息转向定价年底前加息可能性一度超过50%（目前已回落至14%左右）。

2. **1990年历史参照**：
   - 当年面临石油供给冲击，市场同样押注美联储收紧政策
   - 但美联储最终选择降息
   - 市场"严重倾向于加息风险并要求可观的风险溢价，即使美联储最终大幅降息"

3. **核心逻辑**：
   - 当前通胀由油价推动，属于供给侧冲击，而非需求侧过热
   - 美联储主席鲍威尔倾向于在能源冲击背景下维持利率不变并"忽略"其影响
   - 经济增速放缓时，美联储更不会因此收紧货币政策

4. **高盛预测**：
   - 首次降息时点从6月推迟至9月，12月第二次降息
   - 仍维持2026年降息两次的基准判断（节奏延后而非政策转向）
   - 布伦特原油均价3月105美元，4月冲高至115美元，年底前回落至80美元
   - 假设霍尔木兹海峡供应中断持续约六周
   - 美国经济衰退概率从冲突前20%上调至30%

## 对专题的贡献

- **货币政策维度**：为理解2026年美以伊战争期间美联储政策选择提供关键历史参照和分析框架。供给侧冲击vs需求侧过热的区分，是判断政策方向的核心。
- **市场预期维度**：揭示市场往往过度反应加息风险（要求风险溢价），而历史经验显示美联储在供给冲击下更可能降息。
- **资产定价维度**：若市场预期回归降息主线，可能为股债市场带来喘息之机。

## 不确定性/局限性

- 报告基于"霍尔木兹海峡中断六周"的假设，若中断长期化，预测可能失效。
- 未充分考虑2025年特朗普关税政策与能源冲击的叠加效应。
- 1990年与2026年的经济结构差异（如债务水平、全球化程度）未充分讨论。

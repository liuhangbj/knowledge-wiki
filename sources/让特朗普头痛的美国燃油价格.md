# 让特朗普头痛的美国燃油价格

## 核心论点

美以伊战争使国际石油价格史上第四次涨破100美元大关。作为世界最大石油生产国和净出口国，美国国内汽柴油价格涨至2022年以来高位。特朗普政府采取大规模释放战略石油储备、临时修改法律放宽监管、推动暂停征收燃油税等措施大力干预，但收效甚微。3美元/加仑和4美元/加仑是美国民众可接受的价格门槛，长期高于此将对执政政党带来非常大不利影响。

## 关键证据

1. **价格数据（2026年5月18日）**：
   - 美国普通汽油平均零售价：4.490美元/加仑（较一年前低点高42%）
   - 高速公路柴油平均零售价：5.596美元/加仑
   - 西海岸汽油均价：5.61美元/加仑（较去年上涨31%）
   - 加州普通汽油价格均价：6.01美元/加仑（突破6美元关口）
   - 美国柴油价格最高时期：2022年6月20日5.81美元/加仑

2. **价格构成**：
   - 原油成本：占50%左右（布伦特影响大于WTI）
   - 炼油成本：占20%上下
   - 税费：占17%左右
   - 销售费用：汽油占13%，柴油更高（超20%）
   - 国际市场原油价格每变动1美元/桶，美国国内普通汽油零售价格产生2.4美分/加仑上行压力

3. **政府干预措施**：
   - 大规模释放战略石油储备：2026年3月11日宣布释放1.72亿桶，截至5月已释放1.0698亿桶（占计划62.19%）
   - 临时修改法律：依据《国防生产法》要求恢复圣伊内斯油田运营；暂时豁免琼斯法案60天
   - 推动暂停征收燃油税：仍处于讨论阶段，预计很难成为现实
   - 讨论限制美国石油出口：白宫官员与石油协会董事会闭门会议讨论，但官方否认有计划

4. **历史对比**：
   - 2008年6-7月：汽油超4美元/加仑约1个月
   - 2022年3-8月：汽油超4美元/加仑约4个月，其中4.7美元/加仑上下约1个月
   - 2026年3月以来：柴油3月9日大涨至4.859美元/加仑，3月16日涨过5美元

5. **政治影响**：
   - 2026年是美国中期选举年
   - 当前汽油价格与拜登执政时最高水平仅差0.354美元/加仑
   - 柴油仅差0.214美元/加仑
   - 交通支出是美国家庭第二大支出（占16%左右）

## 对专题的贡献

- **能源价格传导维度**：详细展示国际原油价格→美国国内汽柴油价格的传导机制（2.4美分/加仑/美元桶），以及区域差异（西海岸最高、墨西哥湾最低）。
- **政策维度**：特朗普政府干预措施（SPR释放、琼斯法案豁免、燃油税暂停）的效果评估，揭示政策工具局限性。
- **政治维度**：燃油价格与中期选举的关联，以及"3美元/4美元门槛"的政治敏感性，为理解特朗普政策转向（如5月临时协议）提供动机分析。

## 不确定性/局限性

- 文章发表于2026年5月，当时战争仍处于"烂尾"状态，后续油价走势（如协议达成后回落）未涵盖。
- 对SPR释放的长期影响（如储备补充成本、战略安全）分析不足。
- 未充分考虑炼油产能瓶颈（如春季检修、停产维护）对价格的独立影响。

# 中东战事升级，全球能源市场陷入系统性危机

## 核心论点

美以伊战事持续升级，多国油气设施受损，全球能源市场进一步承压。伊朗封锁霍尔木兹海峡已近20天，海湾国家原油日产量削减近1000万桶（约全球供应10%）。布伦特原油价格站稳100美元/桶上方，3月19日盘中触及119美元/桶。危机已从能源向化工、金属市场传导，形成系统性供应链危机。

## 关键证据

1. **石油市场**：
   - 伊朗封锁霍尔木兹海峡近20天，日均约2000万桶的海峡原油运输量大幅下滑
   - 海湾国家原油日产量削减近1000万桶（约全球供应10%）
   - 布伦特原油站稳100美元/桶上方，3月19日盘中触及119美元/桶
   - WTI突破99美元/桶
   - 阿曼原油期货涨至153美元/桶，迪拜、阿曼现货价格攀升至155美元/桶历史高位
   - 花旗预测：若能源基础设施遭广泛袭击、海峡长期关闭，布伦特油价Q2-Q3均价可能达130美元/桶

2. **天然气市场**：
   - 卡塔尔拉斯拉凡工业城遭袭并造成"大面积损毁"，全球约20%LNG生产能力陷入停滞
   - 霍尔木兹海峡承担全球约20%LNG海运贸易量
   - 亚洲JKM现货价格从冲突前10-12美元/百万英热飙升至25美元以上（涨幅超100%）
   - 欧洲天然气4月期货价格上涨逾30%，至每兆瓦时70欧元以上（较战前上涨逾一倍）
   - 美国天然气期货上涨约6%，至每百万英热单位约3.2美元
   - 全球进入"抢货模式"：原本运往欧洲的美国LNG船货改道亚洲

3. **化工市场**：
   - 亚洲石化产业面临直接威胁：去年该地区超90%石油、86%LNG供应通过霍尔木兹海峡
   - 超过200万桶/日亚洲石脑油进口依赖此航道
   - 全球聚合物价格两周内飙升24%-75%
   - 韩国石化巨头（乐天化学、LG化学、YNCC）降低开工负荷，部分装置宣布不可抗力
   - 氯乙烯单体（VCM）涨幅达51.85%，国际石脑油涨幅达38.87%
   - 全球超30%尿素贸易、约20%氨和磷酸盐贸易需经过该海峡

4. **金属市场**：
   - 铝市场反应最剧烈：中东贡献全球约8%铝供应
   - 巴林铝业（Alba）宣布进入不可抗力状态并暂停发货
   - 卡塔尔电解铝厂（Qatalum）维持约60%运行水平
   - LME铝价报约3370美元/吨（较冲突前累计上涨约8%，逼近四年高位）
   - CRU Group警告：若库存持续下滑且供应中断延续，铝价有望攀升至4000美元/吨
   - 镍、铜等矿产提炼受波及：印尼镍生产商75%硫磺原料依赖中东进口

5. **氦气危机**：
   - 韩国2025年65%氦气供应依赖卡塔尔
   - 物流链上约三分之一低温集装箱滞留在封锁区域内
   - 液氦在集装箱内仅能维持约45天
   - 全球高精尖制造业面临严重实物断货风险

## 对专题的贡献

- **系统性危机维度**：首次展示战争从能源→化工→金属→半导体的完整传导链条，揭示"能源危机"实为"工业体系危机"。
- **量化维度**：提供具体数据（1000万桶/日减产、20%LNG产能停滞、聚合物价格飙升24-75%），为后续分析提供基准。
- **区域维度**：亚洲因更高的霍尔木兹依赖度（84%石油、86%LNG）承受更大冲击，与欧美形成对比。

## 不确定性/局限性

- 文章发表于2026年3月20日，当时战事仍在升级，后续发展（如5月临时协议）未涵盖。
- 对设施修复时间（"以年计"）的估计基于专家判断，可能因技术进步或政治意愿而缩短。
- 未充分考虑各国战略储备释放（如IEA 4亿桶）对价格的缓冲作用。

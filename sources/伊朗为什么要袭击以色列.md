# 伊朗为什么要袭击以色列？

## 核心论点

2024年4月13日伊朗对以色列的首次直接袭击，是对4月1日以色列空袭伊朗驻大马士革领事馆的报复。这一事件标志着伊朗从"影子战争"（不承认负责）转向公开直接对抗，是伊朗-以色列关系的历史性"范式转变"。

## 关键证据

1. **袭击背景**：4月1日以色列空袭伊朗驻叙利亚大马士革领事馆，造成13人遇难，包括革命卫队高级指挥官穆罕默德·礼萨·扎黑迪准将。伊朗称此举侵犯其主权。

2. **首次直接攻击**：伊朗使用无人机和巡航导弹对以色列发动首次直接袭击，声称袭击了以色列在尼格夫的一个空军基地。以色列称大多数导弹在到达领土前被拦截。

3. **历史关系**：以色列和伊朗曾是盟友，直到1979年伊斯兰革命后伊朗将反对以色列作为意识形态核心。伊朗最高领袖哈梅内伊称以色列为"毒瘤"。

4. **伊朗盟友网络**：
   - 叙利亚：最重要盟友，伊朗与俄罗斯共同支持阿萨德政府
   - 黎巴嫩真主党：最强大的代理人武装，几乎每天与以色列交火
   - 伊拉克什叶派民兵：袭击美国在伊拉克、叙利亚和约旦的基地
   - 也门胡塞武装：控制也门人口最稠密地区，向以色列发射导弹和无人机
   - 巴勒斯坦武装组织：包括哈马斯，伊朗否认参与10月7日袭击但提供武器和训练

5. **军力对比**：伊朗在导弹和无人机方面投入巨资，拥有庞大武器库；以色列拥有世界最先进的空军之一（14个战斗机中队，包括F-35），以及深入敌国打击经验。

6. **核能力**：以色列被认为拥有核武器；伊朗否认寻求核武器，但福尔多地下基地发现丰度83.7%的浓缩铀颗粒（接近武器级）。

## 对专题的贡献

- **军事维度**：详细记录伊朗从代理人战争转向直接对抗的决策逻辑和军事能力，为理解2026年战争升级提供关键背景。
- **战略维度**：伊朗选择直接而非通过真主党回应，被分析为"积极的"（避免扩大与黎巴嫩冲突），但也展示了伊朗"展示能力但不升级"的精密计算。
- **能源维度**：伊朗对海湾国家油气基础设施的打击能力，以及霍尔木兹海峡封锁威胁，在本文中已有伏笔。

## 不确定性/局限性

- 文章发表于2024年4月，当时冲突尚未全面升级，对后续大规模战争的预测有限。
- 对伊朗内部决策过程（如最高领袖批准袭击的机制）缺乏深入分析。
- 未量化分析冲突对全球能源市场的潜在影响。

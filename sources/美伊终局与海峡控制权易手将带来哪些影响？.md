# 美伊终局与海峡控制权易手将带来哪些影响？

## 核心论点

市场误将"谅解备忘录"等同于"恢复原状"，但霍尔木兹海峡已进入"收费制度化"新常态。文章构建四层涨价传导链（能源→化工→高分子材料→科技农业），识别出氦气物理损毁、乙二醇成本曲线位移、石油美元裂缝等被低估变量，论证战后秩序重塑将系统性削弱美元定价权基础。

## 关键信息

- **海峡新常态**：伊朗议会3月30日通过通行费法案，4月23日首笔200万美元入账，5月5日"波斯湾海峡管理局"挂牌；引入阿曼"共管"
- **四层涨价链**：
  1. 能源链：原油（日本47.3%、韩国35.4%中东依赖）、石脑油、柴油/航煤（欧盟42%依赖）、LNG、LPG（中国丁烷30.3%依赖）
  2. 化工链：乙二醇（中国30.5%、欧盟22.9%、韩国22.1%依赖）、甲醇（中国25%、日本23.5%）、醚醇类（中国44.7%）
  3. 材料链：LLDPE/LDPE（中国26.7%/26.1%）、POE树脂（中国12.2%，光伏封装材料）
  4. 科技农业链：氦气（中国31%、韩国24.5%、日本16.2%、欧盟15.2%依赖），卡塔尔设施物理损毁
- **石油美元裂缝**：伊朗要求通行费以里亚尔支付，若非美元结算落地，体系松动将从象征走向制度
- **盟友离心**：沙特阿美、卡塔尔LNG设施遭袭，美国安全承诺可信度受损；欧洲、日韩可能为保通行权对伊妥协
- **配置建议**：黄金、人民币、有色金属受益；高估值科技股面临中东资金撤离风险；新能源、电力设备、军工制造获需求支撑
